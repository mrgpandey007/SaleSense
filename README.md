# Text-Based Sales Prediction using Machine Learning

## 📌 Problem Statement
In modern e-commerce platforms, large volumes of customer reviews are generated daily. These reviews contain valuable insights about customer sentiment and product perception. The objective of this project is to predict product sales (or demand) based on textual reviews using machine learning techniques.

---

## 📊 Dataset
This project uses the Amazon Customer Reviews dataset.

- Contains product reviews, ratings, and metadata  
- Ratings are used as a proxy for sales performance  
- Source: Kaggle  

---

## 🛠️ Tech Stack

## ⚙️ Project Pipeline

1. Data Collection

6. Model Evaluation  
7. Result Visualization

## 🤖 Models Used
- Linear Regression  
- Random Forest Regressor
